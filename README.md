## SUPER-4
#### Build instruction
1) Make sure you have installed [iverilog](http://iverilog.icarus.com/), make, gtkwave
2) Clone this repo
```
git clone https://github.com/p-m-a-w/super-4.git
```
3) Run this command
```
cd super-4
make simulation
```
4) Open vcd with gtkwave on file ``simulation/game_stim.vcd``
